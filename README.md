- .NET 8.0 SDK or later
- Windows 10/11
- Administrator privileges (for performance counters)

## Building

```bash
dotnet build
```

## Running

```bash
dotnet run
```

## Project Structure

- `App.xaml` - Application resources and global styles
- `App.xaml.cs` - Application code-behind
- `MainWindow.xaml` - Main dashboard UI
- `MainWindow.xaml.cs` - Dashboard logic and performance monitoring

## Color Palette

- **Primary**: `#0D1117` (Dark blue-black)
- **Secondary**: `#161B22` (Darker blue)
- **Accent**: `#00D9FF` (Neon cyan)
- **Accent Alt**: `#FF006E` (Neon magenta)
- **Text**: `#E8F0FF` (Light blue-white)

## Customization

You can easily customize the dashboard by:
1. Modifying colors in `App.xaml` resource definitions
2. Adding new monitoring cards in `MainWindow.xaml`
3. Implementing additional performance metrics in `MainWindow.xaml.cs`
